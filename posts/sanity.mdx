## Purpose

- Let's learn how to display the added and updated content from Sanity Studio by directly applying it through SSR (Server-Side Rendering) in the NextJS 13 App.

- Next, Let's learn how to display the added and updated content from Sanity Studio by applying it during the build time (SSG: Static-Site Generation) in the NextJS 13 App.

## 4. First GROQ Query

Using GROQ Query to fetch data

- Create (next-sanity/sanity/sanity-utils.ts) file

```javascript
import { createClient, groq } from "next-sanity";

export async function getProjects() {
  const client = createClient({
    projectId: "poxxqfti",
    dataset: "production",
    apiVersion: "2023-04-24",
  });

  return client.fetch(
    groq`*[_type == "project"] {
        _id,
        _createdAt,
        name,
        "slug": slug.current, 
        "image": image.asset->url,
        url,
        content,
    }`
  );
}
```

- Modify NextJS Home page : ~/app/page.tsx

```javascript
import { getProjects } from "../../sanity/sanity-utils";
// import project from "../../sanity/schemas/project-schema";

export default async function Home() {
  const projects = await getProjects();

  return (
    <div>
      {projects.map((project) => (
        <div key={project._id}> {project.name}</div>
      ))}
    </div>
  );
}
```

### With the use of SSR instead of SSG, starting from NextJS 13, there is no need to use getStaticPath() and getStaticProps() to fetch Sanity data.

## 5. Mapping over projects

Define Types

- Create (next-sanity/types) directory and "Project.ts" file
- Define Project type in (next-sanity/types/Project.ts) file

```javascript
import { PortableTextBlock } from "sanity";

export type Project = {
  _id: string,
  _createdAt: Date,
  name: string,
  slug: string,
  image: string,
  url: string,
  content: PortableTextBlock[],
};
```

File modification and type application

- Modify (next-sanity/sanity/sanity-utils.ts) file

```javascript
import { createClient, groq } from "next-sanity";
import { Project } from "../types/Project";

export async function getProjects(): Promise<Project[]> {
  const client = createClient({
    projectId: "poxxqfti",
    dataset: "production",
    apiVersion: "2023-04-24",
  });

  return client.fetch(
    groq`*[_type == "project"] {
        _id,
        _createdAt,
        name,
        "slug": slug.current, 
        "image": image.asset->url,
        url,
        content,
    }`
  );
}
```
